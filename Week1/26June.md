## Data Types

- Which type of data is being used in an application (For example, a website for a Gym has a form which will take data from the user)

- is of two types: Primitive and Non-Primitive
- Primitive: have fixed size, temporary storage usage 
- Non-Primitive: linearly expandable size, permanent memory storage
